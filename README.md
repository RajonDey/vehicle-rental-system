# Vehicle Rental System - Database Design & SQL Queries

## Overview
This project is a Vehicle Rental System designed to manage users, vehicles, and bookings. 
It demonstrates understanding of database design (ERD), relationships, primary/foreign keys, and SQL queries.

## SQL Queries
The `queries.sql` file contains the following:

1. Retrieve booking information with customer name and vehicle name (INNER JOIN)
2. Find vehicles never booked (NOT EXISTS)
3. Retrieve available vehicles of a specific type (WHERE)
4. Total bookings per vehicle with more than 2 bookings (GROUP BY, HAVING)
